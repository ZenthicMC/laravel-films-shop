# Films Shop
A Web Application Films Shop With Payment Gateway Using Tripay

# Tech Stack
<ul>
  <li>Laravel 9</li>
  <li>Boostrap 5</li>
  <li>Laravel Breeze</li>
  <li>Omdb API</li>
  <li>Tripay API</li>
</ul>
